# Add –org flag to create issue CLI command

The daemon already supports org-wide issues via the is_org_issue field in CreateIssueRequest, but the CLI’s create issue command doesn’t expose this option. Add an –org flag to the create issue command that sets is_org_issue=true, enabling cross-org issue creation from the CLI.
