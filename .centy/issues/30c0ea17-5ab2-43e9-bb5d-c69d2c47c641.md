# Create lib/create-pr module for interactive PR creation

Create an interactive PR creation library similar to lib/create-issue/ with prompts for title, description, branches, linked issues, reviewers, and priority.

## Acceptance Criteria

- [ ] Create centy-cli/src/lib/create-pr/ directory with:
  - index.ts - exports createPr function
  - create-pr.ts - main orchestration function
  - gather-pr-input.ts - gathers input from options or prompts
  - prompt-for-title.ts - prompts for PR title
  - prompt-for-description.ts - prompts for PR description
  - prompt-for-source-branch.ts - prompts for source branch (with git auto-detection)
  - prompt-for-target-branch.ts - prompts for target branch (default: main)
  - prompt-for-linked-issues.ts - prompts for linked issue numbers
  - prompt-for-reviewers.ts - prompts for reviewer names
  - prompt-for-priority.ts - reuse or adapt from create-issue
- [ ] Create types: create-pr-options.ts, create-pr-result.ts
- [ ] Update commands/create/pr.ts to use new library (make flags optional)
- [ ] Add unit and integration tests

## Reference Files

- centy-cli/src/lib/create-issue/ (pattern to follow)
- centy-cli/src/commands/create/pr.ts (current non-interactive implementation)
