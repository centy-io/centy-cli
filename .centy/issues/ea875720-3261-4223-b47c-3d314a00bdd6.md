# Add `centy next <type>` command

Add a new `next` command that returns the next open item of any type.\n\nUsage:\n centy next issue\n centy next issue --status in-progress\n centy next bug --json\n centy next issue --project centy-daemon\n\nBehavior:\n- Accepts a required `type` arg (pluralized internally)\n- Defaults to --status open\n- Uses daemonListItems with limit:1 and a JSON MQL filter\n- Prints display number, title, status, priority, and body\n- --json flag outputs the raw item as JSON\n- Shows "No <status> <type> found." when empty
