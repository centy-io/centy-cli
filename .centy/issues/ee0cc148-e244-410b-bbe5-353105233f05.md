# Add --global flag to list command for cross-project queries

## Summary

Add a `--global` flag to the `centy list` command that queries items across all tracked projects, returning results sorted by creation date (newest first).

## Requirements

- New `--global` / `-g` flag on the `list` command
- When used, call the new `ListItemsAcrossProjects` daemon RPC instead of `ListItems`
- Display results with project name prefix, e.g. `[centy-daemon] #354 Add cross-project ListItems RPC [open] [P2]`
- JSON output should include `projectName` and `projectPath` fields (use `ItemWithProject` shape)
- Works with existing `--limit`, `--offset`, `--status`, `--priority` filters
- Skip `--project` flag validation when `--global` is set (no project context needed)

## Example usage

```bash
centy list issues --global --limit 1
centy list issues --global --status open --json
```
