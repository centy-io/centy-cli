# Add daemon installation command

Add a centy install daemon command to download and install the centy daemon from GitHub Releases.

## Requirements

- Command: centy install daemon
- Source: GitHub Releases (centy-io/centy-daemon)
- Install path: ~/.centy/bin/centy-daemon
- Platforms: macOS (arm64/x64), Linux (arm64/x64), Windows (x64)

## Flags

- –version, -v: Specific version (default: latest)
- –force, -f: Overwrite existing installation

## Files to Create

1. src/commands/install/daemon.ts - The command
1. src/lib/install/types.ts - TypeScript interfaces
1. src/lib/install/detect-platform.ts - OS/arch detection
1. src/lib/install/github-releases.ts - GitHub API interaction
1. src/lib/install/download-binary.ts - Download with progress
1. src/lib/install/install-daemon.ts - Main orchestrator
1. src/lib/install/index.ts - Re-exports

## Files to Modify

1. src/lib/start/find-daemon-binary.ts - Add ~/.centy/bin/ lookup
1. src/hooks/prerun.ts - Exclude install daemon from daemon check

## Implementation

- Use native fetch API for downloads
- Show progress during download
- Make binary executable on Unix
- No new npm dependencies needed
